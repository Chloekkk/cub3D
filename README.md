## Getting Started 🚩

### 📦 Installation
Clone the repo
   ```sh
   git clone https://github.com/Chloekkk/cub3D.git
   ```

### 🛠️ Build
Build with make command
```sh
make
```

### 🚀 Run
```sh
./cub3D map/kartrider.cub
```
